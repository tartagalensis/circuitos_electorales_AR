# circuitos_electorales_AR
Este repositorio contiene los circuitos electorales de cada provincia/distrito de la República Argentina
Cada distrito está en un archivo geojson que contiene 4 columnas:

* Circuito
* Codprov 
* Coddepto
*  Geometry
